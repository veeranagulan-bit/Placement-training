# ☕ Java Placement Programs

A collection of simple Java programs commonly asked in placement interviews. This repository is created to practice programming, improve problem-solving skills, and prepare for coding rounds.

## 📌 Topics Covered

- Pattern Programs
- Number Programs
- String Programs
- Array Programs
- Searching Algorithms
- Sorting Algorithms
- Recursion
- Functions
- Loops
- Basic Data Structures

## 🛠️ Technologies Used

- Java
- Visual Studio Code

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/java-placement-programs.git
   ```

2. Open the project in Visual Studio Code.

3. Compile the Java file:
   ```bash
   javac ProgramName.java
   ```

4. Run the program:
   ```bash
   java ProgramName
   ```
